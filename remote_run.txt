# to run commands remotely on firstlight

1. edit code (if you need to) and commit and sync with github
2. ssh into firstlight
3. cd into folder that is the github repository
4. >> git pull
5. cd into wherever the code is (or if its just a command you can run from anywhere then you can just do the next step)
eg. assuming you're currently in one of the pears project repositories
>> cd codes/
6. >> nohup command &
>> disown (do this quickly after entering the previous command)
7. logout of firstlight
